About Getodone

Getodone is a next-level AI-powered todo app that doesn’t just track your tasks — it helps you get them done. With built-in LLM-powered motivation, personalized reminders, and user-configurable tone (soft or hard), your productivity just leveled up.

Key Features

  • ✅ Simple todo creation interface
  • AI-based motivational notifications
  • ⚙️ User-configurable personality (soft/hard)
  • API key stays local via AsyncStorage
  • LLM-powered personalized nudges
